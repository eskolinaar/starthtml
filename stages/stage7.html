Schritt 7 - Eine Verknüpfung / Ein Link

Wenn man auf einen Link klickt, wechselt der Browser auf den verlinkten Inhalt.

Dafür verwendet man das A-Tag. Der Inhalt zwischen dem öffnenden und schließenden Tag wird weiterhin angezeigt. Im HREF-Attribut geben wir die Adresse an.

Als Adresse kann man zum Beispiel ein Bild oder eine URL angeben.

Füge folgenden Link an einer Stelle deiner Wahl ein.

<a href="verknüpfte Adresse">Angezeigter Link-Text</a>

Setze die verknüpfte Adresse auf rudi.png
